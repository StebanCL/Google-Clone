# Google Clone UI – Educational Project

A high-fidelity replica of the Google Search interface, developed as a technical challenge to master **UI Architecture** and **Modern CSS Layouts**.

## 📖 Project Purpose
This repository is a **purely educational environment** created by **StebanCL**. The main objective was to explore the intersection between design precision and efficient development using **Vibecoding** methodologies.

## 🚀 Key Features
* **Pixel-Perfect UI**: High-fidelity replication of the search engine's interface.
* **Dark Mode Optimization**: Consistent use of Google’s dark theme color palette.
* **Interactive Components**: Fully functional profile and apps dropdown menus.
* **Vibecoding Workflow**: Leveraging AI-assisted development for rapid iteration.

## 🛠️ Tech Stack
* **HTML5**: Semantic structure.
* **CSS3**: Custom properties, Flexbox, and Grid for a responsive experience.
* **Vanilla JS**: DOM manipulation for interactive elements.
